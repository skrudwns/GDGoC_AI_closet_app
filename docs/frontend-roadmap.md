# Flutter Frontend Roadmap

## Phase 0: Setup

- Install Flutter and Dart.
- Create Flutter app.
- Add linting.
- Add app theme from `DESIGN.md`.
- Add mock data fixtures.

## Phase 1: App Shell

- Bottom navigation:
  - Home
  - Closet
  - Add
  - Ask
  - Profile or Settings
- Shared theme.
- Shared error, loading, empty states.

## Phase 2: Closet

- Clothing grid.
- Filters.
- Item detail.
- Edit metadata.

## Phase 3: Add Item

- Camera/gallery import.
- Upload/classification loading state.
- Classification review screen.
- Save item.

## Phase 4: Ask Closet

- Chat input.
- Recommendation cards with item thumbnails.
- Outfit reason text.
- Swap and save outfit actions.

## Phase 5: Backend Integration

- FastAPI client.
- Multipart upload.
- Auth or device identity if needed.
- Error handling and retry.
- Environment-based API base URL.

## Phase 6: Polish

- Responsive mobile checks.
- Accessibility labels.
- Offline and slow-network behavior.
- Real-device camera testing.
