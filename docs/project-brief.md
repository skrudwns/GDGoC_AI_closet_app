# AI Closet Project Brief

## One-Line Pitch

AI Closet is a mobile wardrobe app that turns clothing photos into structured closet data, then uses that closet data to answer outfit questions.

## Primary Users

- Students and young adults who want quick outfit suggestions.
- Users who own enough clothes that remembering combinations is annoying.
- Users who want AI help but still want control over their closet data.

## Core Jobs

1. Add a clothing item quickly.
2. Confirm or fix AI classification.
3. Browse and filter saved clothing.
4. Ask for outfit recommendations.
5. Understand why an outfit was recommended.

## MVP Scope

- Flutter mobile frontend.
- Camera/gallery item creation flow.
- AI classification loading and review UI.
- Closet grid and item detail.
- Ask Closet chat screen.
- Mock API layer that can be swapped for FastAPI.

## Out of Scope for First Frontend Pass

- Training or serving Fashionpedia.
- Backend deployment.
- Payment, accounts, or social sharing.
- Complex outfit calendar.
- E-commerce links.
